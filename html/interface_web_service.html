Detailed Description

Base class for managing RESTful web service requests.

This class provides a mechanism for sending an HTTP GET or POST request to a RESTful web service and getting the response.

Sends an HTTP GET request to a web service method.

The full URL for the request is in the format: baseUrl\service\method(?query)

The request is sent asynchronously. When complete, the -didGetResponse:orError: method is called.

Parameters:
method The name of the web service method.
