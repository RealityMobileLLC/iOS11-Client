- (void) presentRvModalViewController: (UIViewController<RvModalViewController> *)  viewController
animated: (BOOL)  animated 

Presents a modal view controller.

This is needed to get around a limitation of UIViewController's -presentModalViewController:animated: method which raises an exception if there is already a modal view controller active.

If a modal view controller is already displayed, this method will queue the given view controller and display it when the first one has disappeared.
