Detailed Description

A static class that manages a RealityVision log file.

The Log class provides a wrapper around NSLog and writes log statements to a file.

Todo:
It would probably be better to only write to the System Log database, as NSLog does by default, and then provide a tool to pull RealityVision log statements from the database for emailing/uploading.
